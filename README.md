# Acadex

Minimal Next.js + Supabase base app for building Acadex.

## Current state

- Public landing page at `/`
- Auth routes at `/auth/*`
- Protected app home at `/home`
- Protected listing entry route at `/listings/new`
- Supabase SSR auth setup preserved

## How to run after cloning

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm run dev
```
